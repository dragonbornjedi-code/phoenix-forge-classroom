# Subcategory Method Map

This file fills out the taxonomy with teaching methods and starter lesson directions for every current subcategory.

Method key:
- `EX`: explicit teaching.
- `PB`: play-based learning.
- `DP`: developmental progression.
- `UDL`: multiple access/response options.
- `RS`: retrieval and spaced practice.
- `CRA`: concrete-representational-abstract.
- `INQ`: inquiry/project-based learning.
- `TA`: task analysis.
- `VS`: visual supports.
- `CO`: co-regulation.
- `SN`: social narrative/roleplay.
- `PS`: productive struggle after an easy start.

## Fill-Out Requirements For Every Future Lesson

Each subcategory needs:
- 3 starter lessons.
- 1 movement-heavy lesson.
- 1 tabletop/maker lesson.
- 1 low-demand lesson.
- 1 Student Edition mission translation.
- 1 progress metric.
- 1 support set for visual, sensory, transition, and communication needs.
- 1 extension for high curiosity/high readiness days.
