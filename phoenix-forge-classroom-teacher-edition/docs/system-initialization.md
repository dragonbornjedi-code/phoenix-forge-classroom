# System Initialization

Five-minute morning routine for Teacher Edition.

## Goal
Start the day with connection, body awareness, one mission, and one tiny promise.

## Script
1. Connection: "Good morning. Are you ready for system check?"
2. Energy: "Show me your battery with 1-5 fingers."
3. Body: "Hungry, thirsty, tired, wiggly, or ready?"
4. Mood: "Pick a color: blue, yellow, or red."
5. Mission: choose one small mission for the morning.
6. Promise: choose one tiny promise he can realistically keep.
7. Launch: "Version today is starting. We can patch it anytime."

## Mission Types
- Brain Mission: label, count, sort, read, or guess-and-check.
- Body Mission: balance, carry, stretch, or fine-motor practice.
- Helper Mission: put away, wipe, feed, carry, or prepare.
- Maker Mission: draw, build, repair, or upgrade.
- Kindness Mission: share, ask first, help, or thank.

## Teacher Notes
- If he starts yellow or red, skip the mission and regulate first.
- If he resists, offer two mission choices.
- If he is hungry or tired, handle the body need before teaching.
- Keep the routine short enough that it can succeed on hard days.
- Record only one useful observation.
